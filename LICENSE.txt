4. ANONYMOUS TELEMETRY

The Software sends anonymous usage statistics to the Author once per day
(a "ping") to help improve the Software. Each ping contains only: a random
installation identifier (not linked to Your identity), the Software
version, the operating system name, the panel and browser language codes,
the list of installed game IDs, the number of instances, and a per-game
count of how many times servers were started. No IP address, server name,
password, configuration value, file path, or other personal data is sent.
The receiving server derives an approximate country from the network
connection but does not store IP addresses.

Telemetry is enabled by default and can be disabled at any time, either by
starting the Software with the "-no-telemetry" command-line flag or by
toggling it off in the web panel's Settings. See the README for details.
